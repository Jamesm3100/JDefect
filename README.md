# JDefect

JDefect is a Lua FiveM Script that adds the ability to mark any vehicle as "Defective"

Features include marking a vehicle as defective, checking if a vehicle is defective, and repairing a vehicle.

/defect - Marks a vehicle as defective
/checkdefect - Checks closest vehicle the player is standing next to if it is defective
/fix - Removes the defective marker off the vehicle. Also cleans the vehicle and repairs any damage. Takes 10 seconds and freezes the vehicle during this time.

Installation:
Download the JDefect file
Move JDefect into your FiveM /resources folder
Add "Start JDefect" to your server.cfg file
